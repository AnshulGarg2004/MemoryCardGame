Memory Match Game
=================

Overview
--------
The Memory Match Game is a fun and interactive card-flipping game where the goal 
is to test and improve your memory. Players must find matching pairs of cards 
within the grid. The game tracks moves, time, and allows you to select different 
difficulty levels and themes.

This project is built using only HTML, CSS, and JavaScript â€” no frameworks required. 
It is lightweight, responsive, and works on both desktop and mobile devices.

--------------------------------------------------

Features
--------
1. Difficulty Levels:
   - Easy (4x4 grid)
   - Medium (6x6 grid)
   - Hard (8x8 grid)

2. Themes:
   - Emoji
   - Animals
   - Fruits
   - Tech
   - Shapes

3. Timer Modes:
   - Count Up (measures how long you take)
   - Count Down (challenge mode with a timer limit)

4. Stats and Tracking:
   - Number of moves
   - Elapsed time
   - Best score saved locally (persists between sessions)

5. Design:
   - Modern, clean UI with animations
   - Celebration effect on winning
   - Fully responsive for small screens

--------------------------------------------------

How to Play
-----------
1. Open the game in your browser (see "How to Run" below).
2. Select the difficulty, theme, and timer mode from the top menu.
3. Click "New Game" to start.
4. Flip two cards by clicking on them:
   - If they match, they stay revealed.
   - If they don't match, they flip back over.
5. Continue until all pairs are matched.
6. A win modal will display your stats once the board is cleared.
